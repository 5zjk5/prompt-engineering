# LangGraph 子图示例

本目录包含LangGraph中子图(Subgraph)的使用示例，展示了如何在复杂工作流中组织和管理多个图。

## 目录内容

- `add_persistence.py`: 演示如何为子图添加持久化存储功能
- `father_son_grandson_noShareState.py`: 展示多级子图（祖父-父-子）结构，且每级图拥有独立状态
- `share_state.py`: 演示子图与父图之间的状态共享机制
- `parent_son_grandson.png`: 多级子图结构的可视化流程图

## 子图功能介绍

### 1. 状态共享
子图可以与父图共享部分状态，也可以拥有自己的私有状态。这使得复杂工作流可以模块化，同时保持数据的有效流动。

### 2. 多级嵌套
支持创建多级嵌套的图结构，如祖父图调用父图，父图调用子图，每级图负责特定的功能处理。

### 3. 持久化支持
可以为子图添加独立的持久化存储，适用于多智能体系统中智能体需要跟踪内部消息历史的场景。

### 4. 独立与集成
子图可以作为独立的图运行，也可以集成到父图中作为一个节点使用。
